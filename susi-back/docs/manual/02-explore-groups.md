# 군별 탐색 (가/나/다/군외) 가이드

> **페이지 경로**: `/jungsi/a`, `/jungsi/b`, `/jungsi/c`, `/jungsi/gunoe`

## 📌 이 페이지에서 할 수 있는 것

- **가군/나군/다군/군외** 대학별 정시 전형 탐색
- 4단계 필터링으로 나에게 맞는 대학 찾기
- 환산점수 기반 합격 가능성 확인
- 관심 대학 찜하기

---

## 🎯 군(群)이란?

정시 모집은 **가/나/다군**으로 나뉘어 진행됩니다.

| 군 | 특징 | 주요 대학 예시 |
|---|------|---------------|
| **가군** | 첫 번째 지원 기회 | 서울대, 연세대, 고려대 일부 |
| **나군** | 두 번째 지원 기회 | 연세대, 고려대, 성균관대 일부 |
| **다군** | 세 번째 지원 기회 | 한양대, 중앙대, 경희대 일부 |
| **군외** | 특수목적대학 | 사관학교, 경찰대, KAIST 등 |

> 💡 **핵심**: 각 군에서 **1개 대학**만 지원 가능! (최대 3개 대학 + 군외)

---

## 🚀 사용 방법: 4단계 필터링

### Step 1: 계열 선택

탐색할 계열을 선택하세요.

| 계열 | 설명 |
|------|------|
| **인문** | 문과 계열 (국어, 사회탐구 중심) |
| **자연** | 이과 계열 (수학, 과학탐구 중심) |
| **예체능** | 예술, 체육 계열 |
| **의약학** | 의대, 치대, 한의대, 약대 |

### Step 2: 모집 방식 선택

대학의 모집 방식을 선택하세요.

| 방식 | 설명 |
|------|------|
| **수능 100%** | 수능 점수만으로 선발 |
| **수능 + 면접** | 수능 + 면접 점수 반영 |
| **수능 + 실기** | 예체능 등 실기 포함 |

### Step 3: 수능 최저학력기준 확인

최저등급 충족 여부를 확인하세요.

```
예시) 서울대 인문계열
- 국어, 수학, 탐구(2과목) 중 3개 등급 합 7 이내
- 영어 2등급 이내
- 한국사 3등급 이내
```

| 필터 옵션 | 설명 |
|----------|------|
| **충족** | 내 성적으로 최저 충족 가능 |
| **미충족** | 최저 기준 미달 |
| **없음** | 최저학력기준 없는 대학 |

### Step 4: 상세 정보 조회

필터링된 대학 목록에서 상세 정보를 확인하세요.

#### 표시 정보

| 항목 | 설명 |
|------|------|
| **대학명** | 대학교 이름 |
| **모집단위** | 학과/학부명 |
| **모집인원** | 정시 선발 인원 |
| **내 환산점수** | 나의 계산된 점수 |
| **작년 커트라인** | 작년 합격선 (50%, 70%) |
| **위험도** | 안전/적정/소신/상향 |

---

## 📊 위험도 해석

| 위험도 | 색상 | 의미 | 합격 확률 |
|--------|------|------|----------|
| **안전** | 🟢 초록 | 합격 가능성 매우 높음 | 90% 이상 |
| **적정** | 🔵 파랑 | 합격 가능성 높음 | 70~89% |
| **소신** | 🟡 노랑 | 도전해볼 만함 | 50~69% |
| **상향** | 🔴 빨강 | 합격 어려움 | 50% 미만 |

## ⭐ 관심 대학 찜하기

마음에 드는 대학을 찜해두세요!

### 찜하는 방법

1. 대학 카드에서 **⭐ 아이콘** 클릭
2. 또는 상세 정보에서 **[관심 대학 추가]** 버튼

### 찜한 대학 확인

- **[관심 대학]** 메뉴에서 확인
- 경로: `/jungsi/interest`

## ❓ 자주 묻는 질문 (FAQ)

### Q1: 환산점수가 안 보여요

**A**: 성적 입력을 먼저 완료해주세요.
1. [성적 입력] 페이지 이동
2. 수능/모의고사 점수 입력
3. 저장 후 다시 탐색

### Q2: 가군에서 2개 대학 지원 가능한가요?

**A**: **불가능**합니다.
- 각 군에서 1개 대학만 지원 가능
- 단, 군외 대학은 별도 지원 가능

### Q3: 작년 커트라인은 어디서 확인하나요?

**A**: 대학 상세 정보에서 확인 가능합니다.
1. 대학 카드 클릭
2. 상세 정보 팝업/페이지
3. **[입결 분석]** 탭에서 연도별 커트라인 확인

### Q4: "환산점수 계산 불가" 라고 나와요

**A**: 아래 원인이 있을 수 있습니다:
- 해당 대학의 환산식이 내 선택과목과 맞지 않음
- 필수 과목 미입력 (예: 과탐 필수인데 사탐 입력)
- 시스템 오류 → 새로고침 후 재시도

### Q5: 의대/치대는 어떤 군에 있나요?

**A**: 대학마다 다릅니다.
- 서울대 의대: 가군
- 연세대 의대: 나군
- 고려대 의대: 나군
- **필터에서 "의약학" 선택** 후 확인

### Q6: 군외 대학은 추가 지원인가요?

**A**: 네, **별도 지원**입니다.
- 가/나/다군 외 추가로 지원 가능
- 사관학교, 경찰대, KAIST, UNIST 등
- 단, 각 학교별 일정/자격 확인 필요

## ⚠️ 주의사항

### 1. 커트라인은 참고용

- 작년 커트라인 ≠ 올해 합격선
- 경쟁률, 난이도에 따라 변동
- **참고 자료**로만 활용

### 2. 환산점수 계산 방식

- 대학마다 환산 방식이 다름
- 같은 점수라도 대학별로 다른 환산점수
- **반드시 환산점수로 비교**

### 3. 수능 최저 확인

- 최저 미충족 시 불합격
- 내 등급으로 최저 충족 가능한지 꼭 확인
- 영어/한국사 등급 특히 주의

## 💡 팁

### 효율적인 탐색 전략

1. **안전 1개 + 적정 1개 + 소신 1개** 조합 추천
2. 모든 군을 한 방향(상향)으로 쓰지 않기
3. 관심 대학은 미리 찜해두고 비교

### 빠른 탐색

- 검색창에 대학명/학과명 직접 입력
- 필터 초기화 후 재검색

### 입결 분석 활용

- 50% 컷: 합격자 중 중위권
- 70% 컷: 합격자 중 하위권 (안전선)
- **70% 컷 이상이면 안전권**
