# 경쟁률 히트맵 가이드

> **페이지 경로**: `/jungsi/heatmap`

## 📌 이 페이지에서 할 수 있는 것

- **전국 230개 대학** 경쟁률 지도 시각화
- 실시간 경쟁률 모니터링
- 지역별/상태별 필터링
- 대학 클릭으로 상세 정보 확인

---

## 🗺️ 지도 인터페이스

### 한반도 지도 기반 UI

- 대학 위치가 **점(dot)**으로 표시
- 색상으로 경쟁 상태 구분
- 클릭/터치로 상세 정보 확인

### 상태별 색상 코드

| 색상 | 상태 | 의미 |
|------|------|------|
| 🔴 **빨강** | 과열 | 경쟁률 급등, 주의 필요 |
| 🟢 **초록** | 기회 | 지원 기회, 경쟁률 하락 |
| 🔵 **파랑** | 안정 | 평년 수준 유지 |
| ⚪ **회색** | 보통 | 특이사항 없음 |

## ❓ 자주 묻는 질문 (FAQ)

### Q1: 경쟁률은 어디서 가져오나요?

**A**: 공식 데이터 소스:
- 어디가 (대입정보포털)
- 진학사
- 유웨이
- 실시간 크롤링으로 수집

### Q2: "과열" 기준은 무엇인가요?

**A**: 아래 조건 중 하나 충족 시:
- 전일 대비 경쟁률 +0.5 이상 상승
- 작년 최종 경쟁률 초과
- 지원자 수 급증 (10% 이상)

### Q3: 모바일에서도 볼 수 있나요?

**A**: 네, 반응형 지원합니다.
- 모바일: 세로 스크롤 지도
- 태블릿/PC: 전체 지도 표시

### Q4: 특정 대학을 찾고 싶어요

**A**: 검색 기능을 이용하세요.
- 상단 검색창에 대학명 입력
- 해당 대학 위치로 자동 이동
