# 알림 설정 가이드

> **페이지 경로**: `/jungsi/notifications`

## 📌 이 페이지에서 할 수 있는 것

- **알림 채널** 설정 (카톡, SMS, 푸시, 이메일)
- **알림 유형** 선택 (경쟁률, 커트라인 등)
- 알림 ON/OFF 관리
- 테스트 알림 발송

---

## 📱 알림 채널

### 지원 채널

| 채널 | 설명 | 특징 |
|------|------|------|
| **카카오톡** | 알림톡 발송 | 빠름, 무료 |
| **SMS** | 문자 메시지 | 휴대폰 필수 |
| **푸시 알림** | 앱/웹 푸시 | 즉시 알림 |
| **이메일** | 메일 발송 | 상세 내용 |

### 채널 연결

#### 카카오톡 연결

1. **[카카오톡 연결]** 버튼 클릭
2. 카카오 계정 로그인
3. 알림 수신 동의
4. 연결 완료

#### SMS 연결

1. **[SMS 연결]** 버튼 클릭
2. 휴대폰 번호 입력
3. 인증번호 확인
4. 연결 완료

#### 푸시 알림 연결

1. **[푸시 허용]** 버튼 클릭
2. 브라우저 알림 허용
3. 연결 완료

## 🔔 알림 유형

### 설정 가능한 알림

| 알림 유형 | 설명 | 기본값 |
|----------|------|--------|
| **경쟁률 급등** | +0.5 이상 상승 시 | ✅ ON |
| **안전권 진입** | 내 점수가 안전권 도달 | ✅ ON |
| **위험권 진입** | 안전권에서 이탈 | ✅ ON |
| **추합 현황** | 추가합격 발표 시 | ☐ OFF |
| **마감 임박** | D-1, D-Day 알림 | ✅ ON |
| **일일 요약** | 하루 1회 종합 리포트 | ☐ OFF |

### 세부 설정

| 설정 | 옵션 |
|------|------|
| **경쟁률 변동폭** | ±0.3 / ±0.5 / ±1.0 |
| **알림 시간** | 즉시 / 모아서 (1시간) |
| **관심 대학만** | ON / OFF |

## 🧪 테스트 알림

설정이 올바른지 테스트:

1. **[테스트 알림 발송]** 클릭
2. 선택한 채널로 테스트 알림 수신
3. 수신 확인

## ❓ 자주 묻는 질문 (FAQ)

### Q1: 카카오톡 알림이 안 와요

**A**: 아래 사항 확인:
- 카카오톡 앱 알림 허용 (설정 > 알림)
- 채널 친구 추가 여부
- 차단 목록 확인

### Q2: 알림이 너무 많이 와요

**A**: 세부 설정 조정:
- 변동폭 기준 높이기 (±0.5 → ±1.0)
- 관심 대학만 ON
- 일부 알림 유형 OFF

### Q3: 야간에는 알림 받고 싶지 않아요

**A**: 현재는 시간대 설정 미지원
- 휴대폰 방해금지 모드 활용
- 추후 업데이트 예정

### Q4: 알림 기록을 볼 수 있나요?

**A**: 네, **[알림 로그]** 탭에서 확인:
- 최근 30일 알림 이력
- 발송 채널, 시간, 내용

## 💡 팁

### 추천 설정

**원서 접수 기간**:
- 모든 알림 ON
- 변동폭 ±0.3으로 민감하게

**평상시**:
- 주요 알림만 ON
- 변동폭 ±0.5

### 채널 조합

- **카톡 + 푸시**: 즉시 확인용
- **이메일**: 상세 내용 보관용
