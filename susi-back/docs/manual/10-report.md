# 리포트 생성 가이드

> **페이지 경로**: `/jungsi/report`

## 📌 이 페이지에서 할 수 있는 것

- 지원 현황 **PDF/이미지** 리포트 생성
- 다양한 포함 옵션 선택
- 다운로드, 공유, 인쇄
- 부모님/선생님께 전송

> ⚠️ **유료 서비스**: 정시 서비스 구매자만 이용 가능

---

## 📄 리포트 구성

### 포함 옵션 (선택)

| 옵션 | 설명 | 기본값 |
|------|------|--------|
| **수능 성적** | 과목별 점수, 백분위, 등급 | ✅ |
| **지원 대학** | 가/나/다군 지원 조합 | ✅ |
| **합격 확률** | 대학별 AI 예측 확률 | ✅ |
| **경쟁률 분석** | 현재/예상 경쟁률 | ☐ |
| **전략 추천** | AI 기반 지원 조언 | ☐ |
| **입결 분석** | 과거 3개년 커트라인 | ☐ |

### 리포트 미리보기

- 생성 전 프리뷰 확인
- 옵션 변경 후 실시간 반영
- 페이지 수 표시

## 📥 생성 및 공유

### 생성 형식

| 형식 | 용도 |
|------|------|
| **PDF** | 인쇄, 보관용 |
| **이미지** | SNS 공유, 카톡 전송 |

### 공유 방법

| 방법 | 설명 |
|------|------|
| **다운로드** | 기기에 저장 |
| **카카오톡** | 카톡으로 바로 전송 |
| **이메일** | 메일로 발송 |
| **인쇄** | 바로 인쇄 |

---

## 🚀 사용 방법

### Step 1: 옵션 선택

1. 포함할 정보 체크
2. 미리보기 확인

### Step 2: 형식 선택

1. PDF 또는 이미지 선택
2. 페이지 크기 선택 (A4 권장)

### Step 3: 생성 및 공유

1. **[리포트 생성]** 클릭
2. 생성 완료 후 공유 방법 선택

## ❓ 자주 묻는 질문 (FAQ)

### Q1: 리포트 생성에 시간이 얼마나 걸리나요?

**A**: 약 **5~10초** 소요됩니다.
- 포함 옵션이 많을수록 시간 증가
- 네트워크 상태에 따라 변동

### Q2: 생성한 리포트를 다시 볼 수 있나요?

**A**: 다운로드한 파일로 확인하세요.
- 서버에 리포트 저장되지 않음
- 필요시 재생성

### Q3: 카카오톡 공유가 안 돼요

**A**: 아래 사항 확인:
- 카카오톡 앱 설치 여부
- 팝업 차단 해제
- 모바일에서 시도

### Q4: 부모님께 자동 발송 가능한가요?

**A**: 네, 이메일 발송 기능 이용:
1. 이메일 공유 선택
2. 부모님 이메일 입력
3. 발송 완료

## 💡 팁

### 리포트 활용법

- **부모님 상담**: 합격 확률 설명
- **선생님 상담**: 전략 조언 요청
- **자기 보관**: 지원 기록 정리

### 추천 설정

**부모님용**: 수능 성적 + 지원 대학 + 합격 확률
**선생님용**: 전체 옵션 포함
**보관용**: PDF + 전체 옵션
