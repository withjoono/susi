# 성적 분석 가이드

> **페이지 경로**: `/jungsi/score-analysis`

## 📌 이 페이지에서 할 수 있는 것

- 입력한 수능 성적 **상세 분석**
- 과목별 **표준점수, 백분위, 등급** 확인
- 전체 합계 및 평균 확인
- 과목별 강점/약점 파악

---

## 📊 분석 항목

### 과목별 성적 카드

| 과목 | 표시 정보 |
|------|----------|
| **국어** | 표준점수, 백분위, 등급 |
| **수학** | 표준점수, 백분위, 등급, 선택과목 |
| **영어** | 등급 (절대평가) |
| **탐구1** | 표준점수, 백분위, 등급, 과목명 |
| **탐구2** | 표준점수, 백분위, 등급, 과목명 |
| **한국사** | 등급 |
| **제2외국어** | 표준점수, 백분위, 등급 (선택) |

### 종합 분석

| 항목 | 설명 |
|------|------|
| **표준점수 합계** | 국+수+탐1+탐2 합산 |
| **백분위 평균** | 전 과목 백분위 평균 |
| **등급 평균** | 전 과목 등급 평균 |
| **전국 상위 %** | 전국 수험생 중 위치 |

## 📈 그래프 해석

### 과목별 백분위 차트

```
국어  ████████████████████ 95%
수학  ██████████████████   90%
탐구1 █████████████████    85%
탐구2 ████████████████     80%
```

- 막대가 길수록 상위권
- 90% 이상: 최상위권
- 80~89%: 상위권
- 70~79%: 중상위권

## ❓ 자주 묻는 질문 (FAQ)

### Q1: 백분위는 어떻게 계산되나요?

**A**: 표준점수를 기반으로 전국 석차를 백분율로 환산합니다.
- 백분위 95 = 상위 5%
- 백분위 80 = 상위 20%

### Q2: 등급은 어떻게 나뉘나요?

**A**: 9등급제 (상대평가)
| 등급 | 비율 | 누적 |
|------|------|------|
| 1등급 | 4% | 4% |
| 2등급 | 7% | 11% |
| 3등급 | 12% | 23% |
| 4등급 | 17% | 40% |

### Q3: 점수를 수정하고 싶어요

**A**: [성적 입력] 페이지에서 수정하세요.
- 수정 후 분석 페이지 자동 갱신
- 환산점수도 재계산됨
