# 정시 서비스 사용 가이드 (종합)

> **페이지 경로**: `/jungsi/guide`

## 📌 빠른 시작 가이드

### 처음 사용자를 위한 5단계

```
1️⃣ 회원가입/로그인
      ↓
2️⃣ 성적 입력
      ↓
3️⃣ 대학 탐색 (가/나/다군)
      ↓
4️⃣ 관심 대학 찜하기
      ↓
5️⃣ 모의지원 조합 만들기
```

## ⚡ 자주 묻는 질문 TOP 10

### 1. 환산점수란?

각 대학의 수능 반영 방식에 따라 **변환된 점수**입니다.
- 대학마다 반영 비율, 가산점이 다름
- 같은 점수라도 대학별로 환산점수 다름

### 2. 위험도는 어떻게 계산되나요?

작년 커트라인 대비 내 점수 위치:
- **안전**: 70%컷 이상
- **적정**: 50~70%컷 사이
- **소신**: 50%컷 근처
- **상향**: 50%컷 미만

### 3. 군별 지원 가능 개수는?

각 군에서 **1개 대학**만 지원 가능:
- 가군 1개 + 나군 1개 + 다군 1개 = 최대 3개
- 군외는 별도 (사관학교, 경찰대 등)

### 4. 경쟁률 데이터는 어디서 오나요?

공식 데이터 소스:
- 대입정보포털 (어디가)
- 진학사, 유웨이
- 각 대학 입학처

### 5. 합격 확률은 정확한가요?

AI 기반 **예측값**입니다:
- 과거 데이터 기반 추정
- 100% 정확하지 않음
- 참고 자료로 활용

### 6. 모의지원은 실제 지원인가요?

**아닙니다**. 시뮬레이션입니다:
- 전략 수립용 도구
- 실제 원서는 대학 입학처에서

### 7. 서비스 이용료가 있나요?

**유료 서비스**입니다:
- 무료 체험 가능 (제한적)
- 전체 기능은 구매 후 이용

### 8. 점수 수정이 가능한가요?

네, 언제든 수정 가능:
- [성적 입력] 페이지에서 수정
- 환산점수 자동 재계산

### 9. 알림은 어떻게 받나요?

4가지 채널 지원:
- 카카오톡, SMS, 푸시, 이메일
- [알림 설정]에서 ON/OFF

### 10. 리포트 공유가 가능한가요?

네, 다양한 방법 지원:
- PDF 다운로드
- 카카오톡 공유
- 이메일 전송
- 인쇄
