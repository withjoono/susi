# 모의지원 조합 가이드

> **페이지 경로**: `/jungsi/combination`

## 📌 이 페이지에서 할 수 있는 것

- **가/나/다군 지원 조합** 만들기
- 여러 조합 저장 및 비교
- 조합별 **합격 가능성 분석**
- 최적의 지원 전략 수립

---

## 🎯 모의지원이란?

실제 원서 접수 전에 **가상으로 지원 조합을 구성**해보는 기능입니다.

```
예시) 나의 모의지원 조합

가군: 서울대학교 경영학과 (소신)
나군: 연세대학교 경제학과 (적정)
다군: 성균관대학교 경영학과 (안전)
```

> 💡 여러 조합을 만들어 비교하고, 최적의 전략을 찾으세요!

---

## 🚀 사용 방법

### Step 1: 조합 만들기

1. **[+ 새 조합 만들기]** 버튼 클릭
2. 조합 이름 입력 (예: "상향 지원", "안전 지원")
3. **[생성]** 클릭

### Step 2: 대학 추가하기

#### 방법 1: 관심 대학에서 추가

1. 조합 카드에서 **[대학 추가]** 클릭
2. 군 선택 (가/나/다)
3. 관심 대학 목록에서 선택
4. **[추가]** 클릭

#### 방법 2: 검색으로 추가

1. **[대학 검색]** 클릭
2. 대학명 또는 학과명 입력
3. 검색 결과에서 선택
4. **[추가]** 클릭

### Step 3: 조합 분석 확인

조합이 완성되면 자동으로 분석 결과가 표시됩니다.

| 분석 항목 | 설명 |
|----------|------|
| **종합 합격률** | 3개 대학 중 1개 이상 합격 확률 |
| **군별 위험도** | 각 대학별 안전/적정/소신/상향 |
| **환산점수 비교** | 내 점수 vs 작년 커트라인 |
| **추천 의견** | AI 기반 지원 전략 조언 |

## 📊 조합 구성 원칙

### 추천 조합 유형

| 유형 | 가군 | 나군 | 다군 | 특징 |
|------|------|------|------|------|
| **안정형** | 적정 | 안전 | 안전 | 합격 확률 극대화 |
| **균형형** | 소신 | 적정 | 안전 | 추천! 균형 잡힌 전략 |
| **도전형** | 상향 | 소신 | 적정 | 상위권 도전 |
| **올인형** | 상향 | 상향 | 소신 | 고위험 고수익 ⚠️ |

### 피해야 할 조합

❌ **모두 상향**: 3개 모두 불합격 위험
❌ **모두 안전**: 상향 지원 기회 낭비
❌ **같은 위험도**: 다양성 부족

## 🔄 조합 관리

### 조합 이름 수정

1. 조합 카드 우측 **[⋮]** 메뉴 클릭
2. **[이름 변경]** 선택
3. 새 이름 입력 후 저장

### 대학 변경

1. 변경할 대학 클릭
2. **[변경]** 버튼 클릭
3. 새 대학 선택

### 조합 삭제

1. 조합 카드 우측 **[⋮]** 메뉴 클릭
2. **[삭제]** 선택
3. 삭제 확인

> ⚠️ 삭제된 조합은 복구할 수 없습니다!

## 📈 조합 비교하기

여러 조합을 만들어 비교해보세요.

### 비교 항목

| 항목 | 조합 A | 조합 B | 조합 C |
|------|--------|--------|--------|
| 종합 합격률 | 85% | 72% | 58% |
| 가군 | 서울대(소신) | 연세대(적정) | 서울대(상향) |
| 나군 | 연세대(적정) | 고려대(적정) | 연세대(소신) |
| 다군 | 성균관대(안전) | 한양대(안전) | 고려대(적정) |

### 최적 조합 선택 기준

1. **종합 합격률** 70% 이상 유지
2. **안전 1개 이상** 포함
3. **목표 대학** 포함 여부
4. **학과 선호도** 고려

## ❓ 자주 묻는 질문 (FAQ)

### Q1: 조합은 몇 개까지 만들 수 있나요?

**A**: **제한 없이** 만들 수 있습니다.
- 다양한 시나리오 테스트 가능
- 최종 1개 조합으로 실제 지원

### Q2: 관심 대학에 없는 대학도 추가 가능한가요?

**A**: 네, 가능합니다.
- 검색 기능으로 모든 대학 추가 가능
- 단, 성적 입력 후 환산점수 계산 필요

### Q3: 군외 대학은 조합에 추가할 수 없나요?

**A**: 현재 **군외는 별도 관리**됩니다.
- 가/나/다군만 조합에 포함
- 군외는 [관심 대학]에서 별도 확인

### Q4: 조합의 합격률은 어떻게 계산되나요?

**A**: 아래 방식으로 계산됩니다:
```
종합 합격률 = 1 - (불합격률A × 불합격률B × 불합격률C)

예시)
- 가군 합격률 30%, 나군 60%, 다군 90%
- 종합 = 1 - (0.7 × 0.4 × 0.1) = 97.2%
```

### Q5: 조합을 실제 원서 접수에 반영할 수 있나요?

**A**: 아니요, **참고용**입니다.
- 실제 원서는 각 대학 입학처에서 접수
- 조합은 전략 수립 도구로 활용

### Q6: 같은 군에 2개 대학을 넣을 수 있나요?

**A**: **불가능**합니다.
- 각 군에 1개 대학만 배치 가능
- 실제 입시 규정과 동일

## ⚠️ 주의사항

### 1. 모의지원 ≠ 실제 지원

- 조합은 **시뮬레이션**입니다
- 실제 원서 접수는 별도로 진행
- 접수 기간, 서류 준비 등 확인 필요

### 2. 합격률은 예측값

- AI 기반 예측이므로 **100% 정확하지 않음**
- 작년 데이터 기반 추정
- 올해 경쟁률/난이도 변동 가능

### 3. 경쟁률 변동 주시

- 원서 접수 기간 중 경쟁률 확인
- [경쟁률 히트맵]에서 실시간 모니터링
- 필요시 전략 수정
