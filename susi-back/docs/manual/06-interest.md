## 🚀 사용 방법

### 관심 대학 추가

1. [군별 탐색] 페이지에서 대학 검색
2. 대학 카드의 **⭐ 아이콘** 클릭
3. 관심 대학에 자동 추가

### 관심 대학 삭제

1. 관심 대학 목록에서 대학 선택
2. **[삭제]** 버튼 클릭
3. 삭제 확인

### 모의지원으로 이동

1. 원하는 대학 선택
2. **[조합에 추가]** 버튼 클릭
3. 조합 선택 후 추가

## ❓ 자주 묻는 질문 (FAQ)

### Q1: 관심 대학은 몇 개까지 등록 가능한가요?

**A**: **제한 없이** 등록 가능합니다.
- 단, 실제 지원은 가/나/다군 각 1개씩만

### Q2: 다른 군으로 옮길 수 있나요?

**A**: 군은 **대학에서 정한 것**이므로 변경 불가합니다.
- 같은 대학이라도 학과별로 군이 다를 수 있음

### Q3: 입결 데이터는 어디서 가져오나요?

**A**: 대학 입학처 공식 발표 자료 기반입니다.
- 최근 3개년 데이터 제공
- 일부 대학은 미공개
