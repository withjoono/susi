## ❓ 자주 묻는 질문 (FAQ)

### Q1: 데이터 출처는 어디인가요?

**A**: 공식 데이터 소스:
- 대입정보포털 (어디가)
- 각 대학 입학처
- 진학사, 유웨이

### Q2: 실시간이 정확히 몇 분 단위인가요?

**A**: 약 **5분 단위**입니다.
- 원본 데이터 갱신 주기에 따라 다를 수 있음
- 대학별로 갱신 시점 상이

### Q3: 모집인원보다 지원자가 적을 수 있나요?

**A**: 네, 가능합니다.
- 접수 초기에는 지원자 < 모집인원
- 마감 시점까지 증가
- 미달 학과도 존재
