## 📊 정시 대시보드 (`/jungsi/dashboard`)

> ⚠️ **유료 서비스**: 정시 서비스 구매자만 전체 기능 이용 가능

### 주요 통계 카드 (4개)

| 카드 | 설명 | 예시 |
|------|------|------|
| **종합 합격률** | 관심 대학 전체 합격 확률 | 78% |
| **지원 대학 수** | 관심 등록된 대학 수 | 12개 |
| **전체 백분위** | 내 점수 전국 위치 | 상위 5% |
| **경쟁 과열** | 경쟁률 급등 대학 수 | 3건 |

### 내 지원 대학 현황

| 표시 항목 | 설명 |
|----------|------|
| 대학명 | 관심 등록 대학 |
| 합격 확률 | AI 예측 확률 |
| 변화도 | 어제 대비 변화 (↑↓) |
| 추이 | 최근 7일 그래프 |

### 빠른 메뉴 (6개)

1. **배치 시뮬레이션** → 모의지원
2. **경쟁률 분석** → 경쟁률 페이지
3. **전국 히트맵** → 히트맵 페이지
4. **알림 설정** → 알림 페이지
5. **PDF 리포트** → 리포트 생성
6. **성적 수정** → 성적 입력

### 실시간 알림

| 알림 유형 | 예시 |
|----------|------|
| 경쟁률 급등 | "연세대 경영 경쟁률 +0.5 상승" |
| 안전권 진입 | "성균관대 경제 안전권 진입!" |
| 마감 임박 | "원서 접수 D-3" |

## ❓ 자주 묻는 질문 (FAQ)

### Q1: 대시보드가 안 보여요

**A**: 아래 사항을 확인하세요:
- 로그인 여부 확인
- 정시 서비스 구매 여부 확인
- 성적 입력 완료 여부 확인

### Q2: 합격률이 0%로 나와요

**A**: 성적을 먼저 입력해주세요:
1. [성적 입력] 페이지 이동
2. 수능/모의고사 점수 입력
3. 저장 후 대시보드 새로고침

### Q3: 무료 체험은 어디서 하나요?

**A**: 두 가지 방법이 있습니다:
- 홈페이지 하단 **[무료 체험하기]** 버튼
- `/jungsi/demo` 직접 접속
