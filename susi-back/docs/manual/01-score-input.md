# 성적 입력 가이드

> **페이지 경로**: `/jungsi/score-input`

## 📌 이 페이지에서 할 수 있는 것

- 수능/모의고사 **표준점수** 입력
- 과목별 점수 저장 및 수정
- 입력한 점수로 전국 대학 환산점수 자동 계산

---

## 🚀 사용 방법

### Step 1: 페이지 접속

1. 상단 메뉴에서 **[정시]** 클릭
2. 사이드바에서 **[성적 입력]** 선택
3. 또는 URL 직접 입력: `https://도메인/jungsi/score-input`

> ⚠️ **로그인 필수**: 성적 입력은 로그인한 회원만 가능합니다.

### Step 2: 입력 모드 선택

| 모드 | 설명 | 상태 |
|------|------|------|
| **표준점수 입력** | 수능 성적표의 표준점수 직접 입력 | ✅ 활성화 |
| 원점수 입력 | 원점수로 입력 (자동 변환) | ❌ 준비중 |

현재는 **표준점수 입력** 모드만 지원됩니다.

### Step 3: 과목별 점수 입력

#### 필수 입력 과목

| 과목 | 입력값 | 예시 |
|------|--------|------|
| **국어** | 표준점수 (0~200) | 131 |
| **수학** | 표준점수 (0~200) | 140 |
| **영어** | 등급 (1~9) | 1 |
| **탐구1** | 표준점수 (0~100) | 68 |
| **탐구2** | 표준점수 (0~100) | 65 |

#### 선택 입력 과목

| 과목 | 입력값 | 비고 |
|------|--------|------|
| **한국사** | 등급 (1~9) | 일부 대학 반영 |
| **제2외국어/한문** | 표준점수 | 일부 대학 탐구 대체 |

#### 선택과목 지정

- **수학**: 확률과통계 / 미적분 / 기하 중 선택
- **탐구**: 사회탐구 또는 과학탐구 과목명 선택

### Step 4: 저장 및 계산

1. 모든 필수 과목 입력 완료
2. **[저장하기]** 버튼 클릭
3. 자동으로 **150개 대학 환산점수 계산** 시작
4. 계산 완료 후 결과 페이지로 이동

## ❓ 자주 묻는 질문 (FAQ)

### Q1: 표준점수는 어디서 확인하나요?

**A**: 수능 성적표에서 확인할 수 있습니다.
- 수능 성적표 → 각 과목별 "표준점수" 항목
- 모의고사의 경우 성적표 또는 학교 공지 확인

### Q2: 영어는 왜 등급만 입력하나요?

**A**: 영어는 2018학년도부터 **절대평가**로 변경되어 표준점수가 없습니다.
- 등급(1~9등급)만 존재
- 대부분의 대학에서 등급별 감점 방식 적용

### Q3: 탐구 과목은 몇 개 입력해야 하나요?

**A**: **2개** 입력이 기본입니다.
- 대부분의 대학이 탐구 2과목 반영
- 일부 대학은 1과목만 반영 (자동 계산됨)

### Q4: 제2외국어는 언제 입력하나요?

**A**: 아래 경우에 입력하세요:
- 서울대, 연세대 등 제2외국어 반영 대학 지원 시
- 탐구 1과목을 제2외국어로 대체하고 싶을 때

### Q5: 점수를 잘못 입력했어요. 수정 가능한가요?

**A**: 네, 가능합니다.
1. 성적 입력 페이지 재방문
2. 기존 점수가 자동으로 불러와짐
3. 수정 후 **[저장하기]** 클릭
4. 환산점수가 자동으로 재계산됨

### Q6: 계산에 시간이 얼마나 걸리나요?

**A**: 약 **1~3초** 소요됩니다.
- 150개 대학 × 488개 환산식 계산
- 서버 상태에 따라 약간의 차이 있음

## ⚠️ 주의사항

1. **정확한 점수 입력**
   - 잘못된 점수 → 잘못된 합격 예측
   - 입력 전 성적표 다시 확인

2. **선택과목 정확히 지정**
   - 미적분/기하/확통 선택에 따라 환산점수 달라짐
   - 탐구 과목명도 정확히 선택

3. **저장 전 확인**
   - 저장 후 바로 계산 시작됨
   - 입력값 다시 한번 검토

## 💡 팁

### 빠른 입력 팁

- **Tab 키**: 다음 입력 필드로 이동
- **숫자 직접 입력**: 드롭다운 대신 숫자 입력 가능

### 점수 최적화

- 여러 모의고사 점수 중 **가장 좋은 점수**로 입력
- 수능 전: 6월, 9월 모평 중 높은 점수 사용
- 수능 후: 실제 수능 점수 입력

### 환산점수 재계산

점수 수정 시:
1. **[점수 삭제]** → **[새로 입력]** 권장
2. 또는 직접 수정 후 **[재계산]** 버튼
