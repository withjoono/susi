자동 배포 (권장):
  git push origin main
  → GitHub Actions 자동 실행
  → 3-5분 후 배포 완료

수동 배포:
  npm run build
  firebase deploy --only hosting:production

배포 URL:
  https://g-frontend.web.app
  https://g-frontend.firebaseapp.com
